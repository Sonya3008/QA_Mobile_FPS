**Test Case 1 — Проверка базовой стрельбы**

ID: ..
Название: Стрельба производится при нажатии кнопки огня
Тип: Functional
Приоритет: High

Предусловия:

- Игрок авторизован

- Запущен мультиплеерный матч

- В руках автоматическое оружие

- Есть патроны

Шаги:

1. Зажать кнопку стрельбы

- Стрельба прекращается при отпускании кнопки

- Нет задержки между нажатием и выстрелом

- Нет самопроизвольных выстрелов

Тип: Functional / Multiplayer
Приоритет: Critical

Тип: Multiplayer / Stability
Приоритет: Critical

Предусловия:

**Test Case 6 — Стабильность клиента во время боя**

ID: ..
Название: Проверка стабильности клиента при активном бое
Тип: Stability
Приоритет: Critical

Предусловия:

- Матч с 6+ игроками

- Активная стрельба и взрывы

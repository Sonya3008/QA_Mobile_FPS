**Цель:** обеспечить удобство и читаемость интерфейса во время боя.

**HUD**

- HUD отображается при старте матча в соответствии с дизайном

- Информация читаема на разных разрешениях

- Индикаторы здоровья/патронов обновляются вовремя

- HUD не перекрывает игровой процесс

- Нет визуальных артефактов

**Экран боя**

- Прицел отображается корректно

- Индикаторы попаданий работают корректно

- Отображение урона соответствует фактическому

- Нет миганий элементов UI

- UI корректно реагирует на события боя

**Меню**

- Корректное открытие/закрытие меню

- Нет фризов при навигации

- Корректное отображение настроек

- Изменения применяются без перезапуска

- Возврат в бой без потери состояния

**Адаптация**

- UI корректно масштабируется под разные экраны

- Поддержка портретного/альбомного режимов (если есть)

- Корректная работа на устройствах с вырезами (notch)

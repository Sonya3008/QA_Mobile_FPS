# QA_Mobile_FPS

Данное портфолио демонстрирует мои навыки тестирования мобильных
мультиплеерных FPS-игр на примере жанра шутеров.

Фокус портфолио:
- тестирование геймплея (стрельба, урон, отдача);
- тестирование мультиплеера;
- UI и управление на мобильных устройствах;
- стабильность клиента;
- оформление тестовой документации и баг-репортов.

**Gameplay_Checklist.md**

- Урон соответствует типу оружия

- Хедшоты регистрируются при поподании в голову

- Стрельба работает при разных FPS

- Отдача стабильна

- Попадания не регистрируются через стены

**Multiplayer_Checklist.md**

- Успешное подключение к матчу

- Корректный матчмейкинг

- Отсутствие рассинхронизации игроков

- Сообщение пользователю о потери соединения

- Выход/переподключение без ошибок

**UI_Checklist.md**

- HUD отображается в соответствии с дизайном

- UI не перекрывает управление

- Корректная адаптация под разные экраны

- Отсутствие артефактов

**Controls_Checklist.md**

- Тач-зоны реагируют без пропусков

- Нет ложных нажатий

- Корректная работа при одновременных нажатиях

**Цель:** проверить корректность боевых механик и честность игрового процесса.

**Стрельба**

- Оружие стреляет при каждом нажатии кнопки

- Нет самопроизвольных выстрелов

- Скорострельность соответствует характеристикам оружия

- Стрельба корректно работает при быстром переключении оружия

- Стрельба блокируется при перезарядке

**Урон**

- Урон соответствует типу оружия

- Хедшоты наносят повышенный урон

- Урон по разным частям тела рассчитывается с соответствии с документацией

- Урон суммируется при серии попаданий

- Урон не наносится при промахе

**Хитрег (Hit Registration)**

- Попадания регистрируются при стабильном соединении

- Попадания регистрируются при высоком ping

- Нет «призрачных» попаданий (визуально есть — урона нет)

- Нет регистрации попаданий сквозь стены

- Хитрег корректен при движении стрелка и цели

**Отдача и разброс**

- Отдача соответствует оружию

- Разброс увеличивается при стрельбе очередью

- Разброс корректно уменьшается при остановке стрельбы

- Отдача не сбрасывается при быстром переключении оружия

**Анимации**

- Анимация выстрела соответствует действию

- Анимация попадания отображается корректно

- Нет рассинхронизации между анимацией и уроном
